2. MYSCENEGRAPH06フォルダ
・概要
　アプリケーションタスクパターン（ゲームオブジェクトタスクパターン)
　構造になっています。

・ビルド構成
　Debug   : OpenGL
  Debug20 : OpenGLES20 for IMGSDK
  　　　　　OpenGLES20-APIを使用したアプリを組込み実機上でそのまま動かすため、
　　　　　　（諸事情により色々と削っています）
  Release : メンテナンスしていません。

・ビルド・実行時に必要なLIB等
　-OpenGL関係
　　適宜パスをあわせてください。

　-DBCon ：STDOUTの代わりに使用しています。
　　http://www.vector.co.jp/soft/win95/prog/se046776.html
　　
  -リソース：MGTDataフォルダ以下

・開発環境
 OS      : WinXP HomeEdition SP3
 統合環境: VC++2010 Express
 言語    : C++

↑↓←→　	  :カメラの回転
SHIFT+↑↓←→:カメラ注視点の移動
HOME: 最初のカメラ状態に戻る
